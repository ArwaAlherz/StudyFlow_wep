# StudyFlow

**StudyFlow** is a web application designed to help students organize their study tasks and manage their academic schedule more efficiently.

## Features
- Dashboard overview of study tasks and progress
- Manage subjects and study materials
- Add and track study tasks
- Calendar for scheduling assignments and exams
- Progress tracking
- Notes section for study notes
- Dark mode for comfortable studying

## Benefits for Students
- Helps students stay organized
- Improves time management
- Tracks learning progress
- Supports better study planning

## Technologies Used
- HTML
- CSS
- JavaScript
